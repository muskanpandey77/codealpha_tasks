# Data Visualization Project

## Project Overview
This project focuses on analyzing sales data using data visualization techniques in Python.  
The goal is to identify patterns, trends, and insights from business data.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Tasks
- Data loading and cleaning
- Exploratory data analysis
- Sales analysis by category
- Profit analysis by region
- Discount vs profit relationship
- Data visualization using graphs

## Key Insights
- Certain categories generate higher sales.
- Profit varies significantly across regions.
- Higher discounts can reduce profit margins.
- Visualization helps in better understanding business performance.

## Conclusion
This project demonstrates how data visualization can be used to analyze business data and extract meaningful insights for decision making.
