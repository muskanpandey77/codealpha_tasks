# Student Performance Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a student performance dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Tasks
- Data Cleaning
- Data Inspection
- Statistical Analysis
- Data Visualization
- Finding Insights

## Key Insights
- Reading and writing scores are strongly related.
- Average math scores are lower than reading scores.
- Most students scored in the middle range.

## Conclusion
This project demonstrates basic data analysis and visualization skills using Python.
